The Air Framework is composed of several independent but integrated packages.

## Core Framework

### [air_framework](https://pub.dev/packages/air_framework)

The main package that orchestrates modularity, dependency injection, routing, and security.

## State Management

### [air_state](https://pub.dev/packages/air_state)

The reactive engine. Lightweight, fast, and key-based. It can be used standalone or as part of the full framework.

### [air_generator](https://pub.dev/packages/air_generator)

The build runner annotation processor. It powers the `@GenerateState` pattern and creates the `Flows` and `Pulses` plumbing.

## Tooling

### [air_cli](https://pub.dev/packages/air_cli)

The command-line interface for scaffolding, automation, and project management.

---

> **Note**: All packages follow semantic versioning and are strictly maintained for compatibility.
